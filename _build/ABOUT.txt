WHAT IS THIS APP?
-----------------
CV-Toposheet is an AI-powered desktop application that
automatically extracts and digitizes geographic feature names
from scanned historical topographic maps
(toposheets).

It converts old paper maps which cannot be searched or
analysed digitally into a structured, searchable database
of place names, rivers, roads, villages, elevation markers,
and other geographic features.


WHAT DOES IT DO?
----------------
1. UPLOAD   - Upload a scanned toposheet image (JPG/PNG/TIF)

2. TILE     - Splits the map into smaller tiles for processing

3. OCR      - Reads text from every tile using AI vision
               (Google Cloud Vision or EasyOCR)

4. CLEAN    - Corrects OCR errors using a Large Language Model
               (Groq / Gemini / OpenAI / Claude)

5. DATABASE - Assembles all extracted features into a
               searchable SQLite database

6. QUERY    - Search and export features by name, type,
               district, grid reference, or coordinates


WHO IS IT FOR?
--------------
  * Geography & GIS researchers
  * Historians and archivists
  * Students working with historical maps
  * Survey of India digitization projects
  * Anyone studying colonial-era cartography


IMPORTANT NOTES
---------------
  * Results may contain OCR or LLM errors — always verify
    extracted features against the original source maps
  * Accuracy improves with higher-tier API keys (Gemini / GCV)
  * Keep your API keys secure and do not share them


SYSTEM REQUIREMENTS
-------------------
  * Windows 10 or later (64-bit)
  * 4 GB RAM minimum (8 GB recommended)
  * Internet connection (for AI API calls)
  * At least one API key (Groq free tier recommended)
  * 2 GB free disk space


AFTER INSTALLATION
------------------
  1. Launch CV-Toposheet from the desktop or Start Menu
  2. A browser window opens at http://127.0.0.1:5000
  3. Click [Settings] and enter your API key
     (Groq is free: https://console.groq.com)
  4. Upload a toposheet and click Process
